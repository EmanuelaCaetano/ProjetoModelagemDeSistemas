# Exemplo de Requisitos Não Funcionais 02

## Sistema: E-commerce

### RNF01. Desempenho
O sistema deve carregar a página inicial em até 3 segundos em conexões padrão de internet.

### RNF02. Escalabilidade
O sistema deve suportar aumento de acessos em períodos promocionais sem perda significativa de desempenho.

### RNF03. Segurança
Os dados de autenticação dos usuários devem ser armazenados de forma criptografada.

### RNF04. Disponibilidade
O sistema deve permanecer disponível 24 horas por dia, 7 dias por semana, com índice mínimo de 99,5% de disponibilidade mensal.

### RNF05. Compatibilidade
O sistema deve funcionar corretamente nas versões recentes dos principais navegadores web.
