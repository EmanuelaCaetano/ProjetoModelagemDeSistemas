# Exemplo de Requisitos Não Funcionais 01

## Sistema: Biblioteca Universitária

### RNF01. Desempenho
O sistema deve responder às consultas de livros em até 2 segundos para até 500 usuários simultâneos.

### RNF02. Disponibilidade
O sistema deve estar disponível para uso durante 99% do tempo no período letivo.

### RNF03. Usabilidade
A interface deve permitir que um usuário realize a busca por livros com no máximo 3 cliques a partir da tela inicial.

### RNF04. Segurança
O sistema deve exigir autenticação para acesso às funções de empréstimo, devolução e cadastro.

### RNF05. Backup
Os dados do sistema devem ser copiados diariamente para evitar perda de informações.
