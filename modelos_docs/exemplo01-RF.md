## Sistema de Biblioteca

### RF01. Cadastrar livro
O sistema deve permitir que o bibliotecário cadastre um novo livro, informando título, autor, ISBN, categoria e ano de publicação.

### RF02. Registrar empréstimo
O sistema deve permitir registrar o empréstimo de um exemplar para um aluno previamente cadastrado.

### RF03. Registrar devolução
O sistema deve permitir registrar a devolução de um exemplar emprestado.

### RF04. Consultar disponibilidade
O sistema deve permitir consultar se um livro possui exemplares disponíveis para empréstimo.
