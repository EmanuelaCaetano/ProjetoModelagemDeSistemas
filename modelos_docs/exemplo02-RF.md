## Sistema Acadêmico

### RF01. Realizar matrícula
O sistema deve permitir que o aluno se matricule em disciplinas disponíveis no período letivo.

### RF02. Consultar boletim
O sistema deve permitir que o aluno consulte suas notas e frequência por disciplina.

### RF03. Lançar notas
O sistema deve permitir que o professor registre notas dos alunos em suas turmas.

### RF04. Emitir histórico escolar
O sistema deve permitir gerar o histórico escolar do aluno com as disciplinas cursadas e seus resultados.

---

## Observação

Requisitos funcionais descrevem **o que o sistema deve fazer**.  
Eles representam serviços, comportamentos e funcionalidades esperadas pelos usuários.
