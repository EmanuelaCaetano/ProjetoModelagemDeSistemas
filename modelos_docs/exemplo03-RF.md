## Sistema de E-commerce

### RF01. Cadastrar produto
O sistema deve permitir que o administrador cadastre produtos com nome, descrição, preço, estoque e categoria.

### RF02. Adicionar produto ao carrinho
O sistema deve permitir que o cliente adicione produtos ao carrinho de compras.

### RF03. Finalizar pedido
O sistema deve permitir que o cliente finalize a compra informando endereço de entrega e forma de pagamento.

### RF04. Consultar status do pedido
O sistema deve permitir que o cliente acompanhe o status de seus pedidos.
